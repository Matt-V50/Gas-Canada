# ⛽ Waterloo Gas Price Chart

Gas price tracking dashboard for Waterloo, Ontario. Data sourced from [GasBuddy](https://www.gasbuddy.com/).

## Features

- **Price trends** — line chart showing gas price history across local stations
- **Time range** — view last 7 days, 1 month, 3 months, or all history
- **Filter by station** — compare all stations or focus on one
- **Filter by brand** — Shell, Esso, Costco, etc.
- **Fuel grade** — Regular, Mid-grade, Premium (multi-select, different line styles)
- **Dark mode** — auto-follows system preference
- **Mobile friendly**

Data updates automatically from Google Sheets. Adding a new station requires no code changes.
